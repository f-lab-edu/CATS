## 개발 및 실행 환경 구축
1. git repository clone
```bash
git clone https://github.com/f-lab-edu/CATS.git
cd CATS
```

2. 프로젝트 의존성 설치
```bash
poetry install
```
4. 가상환경 활성화
```bash
poetry shell
```
